# Security Policy

## Supported Versions

We actively support and apply security updates to the following versions of Calamo:

| Version | Supported |
| ------- | --------- |
| main    | Yes       |

## Reporting a Vulnerability

If you discover a potential security vulnerability in Calamo, please do not report it using public issues. Instead, please report it via the contact methods listed below or by creating a private vulnerability report on GitHub if available.

Please include:
- A detailed description of the vulnerability.
- Steps or proof-of-concept scripts to reproduce the issue.
- Potential impact of the vulnerability.

We will acknowledge receipt of your report and work to resolve the issue promptly.
